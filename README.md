# QA Portfolio: Healthcare & Enterprise Systems

Welcome to my professional QA portfolio. This repository contains samples of test documentation and artifacts developed for complex web-based systems.

## 📂 Portfolio Structure

- [Test Documentation](./test-documentation/) — Test Cases (exported from Qase.io) and structured Checklists.
- [API Testing Samples](./api-testing/) — Postman collections with basic tests for REST API validation.
- [SQL Queries](./sql-queries/) — Scripts for back-end data verification and integrity checks.
- [Bug Reports](./bug-reports/) — Detailed bug reports with steps, severity, and expected results.
